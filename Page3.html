Vad är en regel i CSS och vad består den av?

En CSS-regel allmänt känd som en stilregel är en instruktion som talar om för webbläsaren hur man presenterar ett element på en webbsida. Den består av.selektorer och ett deklarationsblock.

Vad är skillnaden mellan id och class? När bör man använda id respektive class?

Id och Class är två attributer med olika funktioner.Nyckelskillnader:

- Unikhet: id används för att specificera element som bara förekommer en gång, till exempel kan det användas för att sätta en rubrik för en given sida. Men klass används om det finns flera saker som delar vissa funktioner (t.ex.)

- Syntax: id börjar med '#' medan klass börjar med '.' när de definieras i HTML-fil.

- Användning: Id används för att identifiera ett enskilt objekt på en sida medan class kan användas flera gånger beroende på om du vill att fler objekt ska se lika ut.
